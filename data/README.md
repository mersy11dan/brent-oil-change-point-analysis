# Data

This directory holds the project data. Large or derived files are git-ignored;
the source price series and the curated events dataset are tracked.

```
data/
├── raw/                  # Source data (immutable)
│   ├── BrentOilPrices.csv    # Daily Brent price (tracked)
│   └── key_events.csv        # Curated key events (tracked)
└── processed/            # Cleaned/derived data (generated, ignored)
    └── brent_clean.csv       # Output of `python -m src.data_loader`
```

## `raw/BrentOilPrices.csv`

Historical daily Brent crude oil spot price.

| Column | Type | Description |
|--------|------|-------------|
| `Date` | date | Trading date. **Mixed formats** in the source: `20-May-87` and `Nov 09, 2022`. |
| `Price` | float | Brent price in USD per barrel. |

- Coverage (observed): 1987-05-20 to 2022-11-14, ~9,011 trading days after cleaning.
- Only trading days are present (gaps on weekends/holidays are expected).

## `raw/key_events.csv`

Curated set of 15 major events used to interpret detected change points.

| Column | Description |
|--------|-------------|
| `event_date` | Approximate start date of the event (YYYY-MM-DD). |
| `event_name` | Short name of the event. |
| `category` | One of: Conflict, OPEC, Economic, Sanctions, Pandemic. |
| `description` | One-line description of the event. |
| `expected_impact` | Qualitative expected effect on price. |

Event dates are approximate; markets may anticipate or lag the official date.

## `processed/brent_clean.csv`

Generated by `python -m src.data_loader`. Cleaned, de-duplicated, chronologically
sorted price series with added `LogPrice` and `LogReturn` columns. Safe to delete
and regenerate.
