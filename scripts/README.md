# Scripts

Standalone, runnable scripts that orchestrate the reusable logic in `src/`.

| Script | Purpose |
|--------|---------|
| `build_report.py` | Generates the self-contained interim HTML report (`reports/interim_report.html`) with embedded figures and the key-events table. |

## Usage

Run scripts from the repository root, e.g.:

```bash
python scripts/build_report.py
```
